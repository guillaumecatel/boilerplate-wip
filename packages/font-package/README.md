# @myorg/fonts



## Installation

```bash
pnpm add @myorg/fonts
```

## Usage

Import fonts in your project:

```typescript
import '@myorg/fonts/font-name.woff2'
```

## Available Fonts

Place your font files in the `src/` directory. Supported formats:
- `.woff2`
- `.woff`
- `.ttf`
- `.otf`
- `.eot`

## Development

```bash
# Copy fonts to dist
pnpm dev

# Build for production
pnpm build

# Clean build artifacts
pnpm clean
```
